# CRM Automation Tests

This repository contains automated tests for verifying product listing functionality on an e-commerce platform.

## Features Tested
1. Product Status Active
2. Product Not Listed After Deletion

## Project Structure
crm-automation/
├─ tests/
│   └─ test_product_listing.py
├─ pages/
│   └─ products_page.py
├─ requirements.txt
├─ README.md
└─ .gitignore

## Setup Instructions
1. Clone the repo:
   git clone https://github.com/praty25sep/crm-automation.git
   cd crm-automation
2. Install dependencies:
   pip install -r requirements.txt
3. Run tests:
   pytest tests/
